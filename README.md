# SELinux-Web-Manager
Conception, développement et déploiement d’un gestionnaire web sécurisé pour l’administration des politiques SELinux avec simulation d’attaques et mécanismes de défense
Fonctionnalités attendues
Visualisation des contextes SELinux 
Activation / désactivation des modes :
enforcing 
permissive 
Gestion des règles 
Logs de sécurité (audit) 

Partie attaque (obligatoire)
exploitation d’une application vulnérable 
accès interdit sans SELinux 

Partie défense
configuration SELinux 
blocage de l’attaque
